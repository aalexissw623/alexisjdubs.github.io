The Essentials of every HTML Page

Every HTML page needs to start with a , which specifies the document and tells the page that it is a HTML file. After, there is the <html> start tag and the </html> end tag at the end of the HTML document. In between, there is the <body> which contains the content on the page. There is a <title>, which represents the title of the page. Then, you can add more in the body like headings using the <h1>, <h2>, <h3>, <h4>, <h5>, <h6>.

HTML <html>

<html> defines the file document and tells the browser it is an HTML file. The tag is the container for all other HTML documents except for the docstring.

Syntax: <html>

Example: <html> <h1>HTML</h1> </html>

Rendering: None

HTML Tag

Header <header>

The tag for headers is <header>. The header element is a container for all of the headings and other navigational tabs or links.

Syntax: <header>

Example: <header> <h1>HTML</h1> </header>

Rendering: None

HTML Header

Title <title>

The <title> titles the name of the tab of the document. It provides a title for the page when it gets saved to favorites.

Syntax: <title>

Example: <title>HTML Reference Title</title>

Rendering: Writes the title in the tab

HTML Title

Body <body>

The <body> defines the body of the document, which includes text, images, hyperlinks, lists, and tables.

Syntax: <body>

Example: <body>The text goes in the body.</body>

Rendering: This is text in the body

HTML Body

Headings <h1>, <h2>, <h3>, <h4>

There are six heading sizes with <h1> being the largest and <h6>.

Syntax: <h1>, <h2>, <h3>, <h4>, <h5>, <h6>.

Example: <h1>This is a heading.</h1>

Rendering:

This is a heading.

HTML Headings

Paragraph <p>

A paragraph is specified by <p>. There is a default margin that is added in front of <p> by browsers.

Syntax: <p>(Add text here)</p>

Example: <p>This is an example.</p>

Rendering: This is a paragraph.

HTML Paragraphs

Breaks <br>

Breaks are used to separate paragraphs, headers, or any blocks of text from each other, or moves the text onto the next line.

Syntax: <br>

Example: <p>Here is some text. <br> Here is more text.</p>

Rendering: Here is some text.
Here is more text.

HTML Breaks

Lines <hr>

The tag <line> is used to draw a line to separate text to make it more visually appeaing. Lines are used to define a thematic change in the content.

Syntax: <hr>

Example: Here is some text. <hr> Here is more text.

Rendering: Here is some text.

Here is more text.

HTML Lines

Comments <!-- -->

Comments are used in text to inform the reader or remind yourself about your code. Comments are not written and do not get displayed in the browser. Comments can be especially useful if there is a lot of code.

Syntax: <!-- -->

Example: <!--This is a comment about a comment in my code.-->

Rendering: None, because they are not displayed in the browser.

HTML Comments

Links <a>

Links, or hyperlinks, are defined by <a>. They link from one webpage to another, so the href attribute is very important. The href attribute is essential because it indicates the destination of the link.

Syntax: <a>

Example: <a href="https://www.w3schools.com">This is a link to the w3schools page.</a>

Rendering: W3Schools Link


HTML Links

Lists <ul>, <ol>, <li>

Lists can be very used to display brief and concise information. Lists may be unordered or ordered. Unordered lists can be bulleted, circles, squares, or have nothing. Ordered lists can be numbers, uppercase letters and lowercase letters, and uppercase and lowercase Roman Numerals.

Syntax: <ul> (unordered list), <ol> (ordered list), <li> (list items)

Example: <ul> <li>Daisy</li>
<li>Tulip</li>
<li>Rose</li> </ul>
