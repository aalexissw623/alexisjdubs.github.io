The Essentials

Every HTML page needs to start with a <!DOCTYPE html>, which specifies the document and tells the page that it is a HTML file. After, there is the <html> start tag at the beginning and the </html> end tag at the end of the HTML document. In between, there is the <body>, which contains the content on the page.

There is a <title>, which represents and changes the name of the tab. More attributes can be added in the body. These attributes can include headings using the <h1>, <h2>, <h3>, <h4>, <h5>, <h6>, paragraphs which are defined by <p>, and other means to display information.

Text can be styled and referenced using CSS, or Cascading Style Sheets. CSS can be referenced and can be very useful in making the website more visually appealing. The distinguishing factor between HTML and CSS is that HTML is mainly used for content and CSS is used for presentation. CSS can be added inline using <div style>, internally ysing a <style> section still in the HTML, and externalling in a separate CSS file. When CSS is added internally, all of the HTML is in its own style section. The internal method is better for one HTML page. It is better to add CSS externally, which is referenced using a link to the file in the header section using an <href>, when the website has multiple pages and a style needs to be changed.

HTML <html>

Description

<html> defines the file document and tells the browser it is an HTML file. The tag is the container for all other HTML documents except for the docstring.

Syntax

Start tag:<html>
End tag: </html>

Example

<html> <h1>HTML</h1> </html>

HTML Document Tag

Header <header>

Description

The tag for headers is <header>. The header element is a container for all of the headings and other navigational tabs or links.

Syntax

Start tag:<headers>
End tag: </headers>

Example

<header> <h2>HTML</h2> </header>

Title <title>

Description

The <title> titles the name of the tab of the document and webpage. It also provides a title for the page when it gets saved to favorites, is bookmarked, or is displayed in history.

Syntax

Start tag:<title>
End tag: </title>

Example

Paragraph <p>

Description

A paragraph is specified by <p>. There is a default margin that is added in front of <p> by browsers. In paragraphs, the browser will automatically strip any extra spaces and extra lines when the page is displayed. Furthermore, different sized screens and windows will create different results.

Syntax

Start tag:<p>
End tag: </p>

Example

<p>This is an example.</p>

Rendering

This is a paragraph.

Description

Breaks are used to separate paragraphs, headers, or any blocks of text from each other, or moves the text onto the next line.

Syntax

Start tag:<br>
End tag: None

Rendering

Here is more text.
The break brings the paragraph onto a new line.

Comments <!-- -->

Description

Comments are used in text to inform the reader or remind yourself about your code. Comments are not written and do not get displayed in the browser. Comments can be especially useful if there is a lot of code.

Syntax

Start tag:<!--
End tag: -->

Example

<!--This is a comment about a comment in my code.-->

Rendering

None, comments are not displayed in the browser.

Lists <ul>, <ol>, <li>

Description

Lists can be very used to display brief and concise information. Lists may be unordered or ordered. Unordered lists can be bulleted, circles, squares, or have nothing. Ordered lists can be numbers, uppercase letters and lowercase letters, and uppercase and lowercase Roman Numerals.

Syntax

Start tag:<ul> (unordered list), <ol> (ordered list), <li> (list items)
End tag: </ul>, </ol>, </li>

Example

<ul>
<li>Daisy</li>
<li>Tulip</li>
<li>Rose</li>
</ul>

Rendering

Blocks <div>

Description

Blocks are the default display value for most elements which are declared using <div>. A block-level element always starts on a new line and takes up the full width of the screen. The <div> is a block-level element and starts on a new line, but the <span> element is an inline element, so it will not start on a new line. The <div> element is often used as a container for other HTML elements, while the <span> element is used for text. Both elements have no required attributes, but both style and class are common.

Syntax

Start tag:<div> (block-level elements), <span> (inline elements)
End tag: </div>, </span>

Example

<div>Hello world</div>
<span>Hello world</span>

Rendering

Block-level elements:
Hello
World

Inline elements:
Hello World
