Exercises: Blocks

Blocks are the default display value for most elements. A block-level element always starts on a new line and takes up the full width of the screen.

Here are some block-level elements < div >:

Hello, hello
World

In the above, the < div > is a block-level element and starts on a new line. The < div > element is often used as a container for other HTML elements. The < div > element has no required attributes, but both style and class are common.

Hello, hello Goodbye

The < span > element is an inline element, so it will not start on a new line. The < span > element is often used as a container for text. Similarly, it has no required attributes, but both style and class are common.
